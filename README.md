# Portfolio van Patricia Podhradska
(NL)
Dit is de vernieuwde versie van mijn eerdere project, 'ParsaDesigns'. Waar de vorige versie de basis legde, tilt deze update mijn digitale visitekaartje naar een hoger niveau.

De website is volledig herbouwd in Next.js en gestyled met TailwindCSS en Shadcn bibliotheek wat zorgt voor een razendsnelle en moderne gebruikerservaring. 
Naast een verfijnd visueel design is de content uitgebreid en de code-architectuur geoptimaliseerd voor betere prestaties en schaalbaarheid. 

Dit project weerspiegelt mijn groei in zowel front-end development als UI/UX design.


(EN)
This is the revamped version of my previous project, 'ParsaDesigns'. While the original version laid the foundation, this update takes my digital business card to the next level.

The website has been completely rebuilt using Next.js and styled with TailwindCSS and the Shadcn library, ensuring a lightning-fast and modern user experience. In addition to a refined visual design, the content has been expanded and the code architecture optimized for better performance and scalability.

This project reflects my growth in both front-end development and UI/UX design.
